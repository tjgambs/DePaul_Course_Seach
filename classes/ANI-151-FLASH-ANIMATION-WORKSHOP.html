ANI 151 FLASH ANIMATION WORKSHOP

This class will introduce students to the fundamentals of creating animation using Flash, including tweening and frame-by-frame drawn animation. This class does not cover Action Script programming. Topics and techniques covered during lectures will be reinforced through in-class exercises and projects.
