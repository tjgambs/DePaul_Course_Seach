CSC 243 PYTHON FOR PROGRAMMERS

An accelerated course covering the essentials of programming, with a focus on problem-solving, structured programming, and algorithm design. The concepts covered include collection types, conditional and iterative structures, functions, file input/output, exceptions, namespaces, recursion, and Internet client programming. PREREQUISITE(S): MAT 130 or equivalent
