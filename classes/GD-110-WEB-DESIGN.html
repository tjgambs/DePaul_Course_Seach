GD 110 WEB DESIGN

An introduction to design process and strategy for the web. Students will be introduced to layout and composition, typography, color, and navigation concerns specific to web design. Projects will be created using both visual design applications and hand-coding with HTML and CSS. The class will also introduce basic components of hypertext transfer protocol, and web site deployment. PREREQUISITE(S): GD 109 or HCI 201
